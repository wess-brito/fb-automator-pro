# FB Automator Pro

Extensão avançada para automatizar convites no Facebook, permitindo convidar usuários que reagiram aos seus posts para curtir sua página de forma segura e eficiente.

## Funcionalidades
- **Auto-Invite**: Convida automaticamente quem reagiu aos posts.
- **Pausa e Retomada**: Controle total sobre a execução da automação.
- **Segurança**: Limites diários para proteção da conta.
- **Logs em Tempo Real**: Acompanhe o progresso diretamente no popup.

## Como Usar locally

**Prerequisites:** Node.js

1. Instale as dependências:
   `npm install`
2. Rode o app para desenvolvimento:
   `npm run dev`
3. Para usar como extensão, carregue a pasta do projeto no Chrome via `chrome://extensions` em modo desenvolvedor.
