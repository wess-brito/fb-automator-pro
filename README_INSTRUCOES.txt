
GUIA DE INSTALAÇÃO - FB AUTOMATOR PRO

1. Crie uma pasta chamada "fb-automator-extensao".
2. Salve os arquivos index.html, index.tsx e manifest.json dentro dela.
3. Use o arquivo icon_source.svg para gerar 3 PNGs:
   - icon16.png (16x16)
   - icon48.png (48x48)
   - icon128.png (128x128)
   (Dica: Você pode usar sites como 'SVG to PNG converter' online).
4. No Chrome, acesse: chrome://extensions
5. Ative o "Modo do desenvolvedor" (canto superior direito).
6. Clique em "Carregar sem compactação" e selecione a pasta que você criou.

PRONTO! Agora basta abrir o Facebook, abrir a lista de reações de um post e clicar no ícone da extensão.
